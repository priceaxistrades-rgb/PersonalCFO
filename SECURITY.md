# Security Policy — Personal CFO

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 5.5.x   | ✅ Current         |
| < 5.0   | ❌ Not supported   |

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability, please report it responsibly.

### How to Report

**Preferred method**: Email `security@personal-cfo.app` with:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will acknowledge receipt within **48 hours** and aim to provide a fix timeline within **7 days**.

### Responsible Disclosure

- Do not disclose the vulnerability publicly until we have released a fix
- We will credit reporters (unless they prefer to remain anonymous)
- We offer a **bug bounty** for critical findings (details upon request)

### What We Consider Critical

- Authentication bypass
- Data leakage / IDOR
- Remote code execution
- SQL injection
- Session hijacking
